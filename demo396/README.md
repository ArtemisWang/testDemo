将一整张专辑的歌曲文件，切分成为一首首单曲


首先将.ape转化为能被苹果电脑识别的.m4a


其次，每首歌曲


1. 按照.cue文件中的title设置文件名

2. 按照INDEX 01的时间作为开始时间

3. 按照下一首歌INDEX 01的时间作为结束时间

如果没有上一首歌代表是第一首，开始时间默认00:00:00

如果没有下一首歌代表是最后一首歌曲，结束时间默认为7分钟，可以认为最后一首歌曲没有超过七分钟的
